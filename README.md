# Heaven of Africa — Система автоматизации контента

**Автоматический конвейер создания контента для африканского продуктового магазина в Дубае.**  
Сайт проекта → [heaven-of-africa-demo.netlify.app](https://heaven-of-africa-demo.netlify.app)

---

## Проблема

Владельцы небольших специализированных магазинов тратят часы на ручное написание описаний продуктов, постов для соцсетей и рецептов — каждую неделю, для каждого нового товара. Владельцу Heaven of Africa (магазин африканских продуктов в Дубае) нужен был способ генерировать готовый контент из каталога продуктов — без ручного написания текстов.

---

## Для кого

- Владельцы и маркетологи специализированных продуктовых магазинов
- Все, кто ведёт каталог товаров и регулярно создаёт контент для соцсетей
- Система не привязана к одному магазину — достаточно заменить `data/products.json` на любой другой каталог

---

## Что умеет проект

### 1. Демо-сайт магазина
Полноценный каталог продуктов:
- 12 африканских продуктов с описаниями, ценами, происхождением и тегами
- **AI-ассистент Amara** — отвечает на вопросы покупателей о продуктах и рецептах в реальном времени (OpenAI API через serverless-функцию Netlify)
- **Модалка с рецептом** на каждой карточке товара — кнопка 📖 открывает полный рецепт с ингредиентами и шагами
- Фильтрация по категориям и поиск по продуктам

### 2. Агенты Claude Code
Четыре агента, которые запускаются из **терминала через Claude Code** — не через сайт. Каждый агент читает каталог продуктов и сохраняет результат в файл на диске.

| Агент | Что делает |
|---|---|
| `recipe-writer` | Читает `products.json`, генерирует по одному рецепту на каждый продукт, сохраняет в `reports/recipes.json` |
| `social-copywriter` | Пишет черновики постов для Telegram и Instagram, сохраняет в `post/` |
| `product-researcher` | Извлекает структурированные данные о продуктах из сайта и JSON — используется как первый шаг пайплайна |
| `image-checker` | Проверяет релевантность фотографий продуктов, формирует отчёт |

### 3. Скилл контент-конвейера
`store-content-pipeline` — запускается одной командой и оркестрирует цепочку: product-researcher → social-copywriter → валидация → сохранение итогового JSON в `post/`.

### 4. Генератор постов (веб-приложение)
Это **отдельный инструмент** — локальное Flask-приложение с интерфейсом в браузере. В отличие от агентов, здесь владелец сам выбирает продукт, платформу и стиль поста и получает результат прямо на экране.
- Выбор продукта из каталога, аудитория, платформа (Telegram / Instagram) и стиль поста
- Четыре стиля: Product · Cultural · Recipe · Lifestyle
- Результат: заголовок, вступление, структура, CTA, хэштеги — редактируются и копируются
- Чек-лист качества с подсчётом символов и проверкой соответствия платформе

## Формат результатов

- `post/african-store-posts.json` — полный вывод контент-конвейера (4–6 постов)
- `post/heaven-of-africa-social.json` — пост по одному продукту (Telegram + Instagram)
- `reports/recipes.json` — 12 рецептов для всех продуктов каталога
- Веб-интерфейс `http://127.0.0.1:5001` — интерактивный генератор постов

## Технологии

- **Python 3 / Flask** — веб-приложение генератора постов
- **HTML / CSS / JavaScript** — сайт магазина и интерфейс генератора
- **Netlify** — хостинг статического сайта + serverless-функции
- **Netlify Functions (Node.js)** — бэкенд AI-ассистента Amara
- **OpenAI API** — ответы Amara
- **Claude Code** — оркестрация агентов, контент-конвейер, генерация рецептов
- **JSON** — каталог продуктов и хранение сгенерированного контента

## Как запустить генератор постов

```bash
cd post-generator
python3 app.py
```

Открыть: [http://127.0.0.1:5001](http://127.0.0.1:5001)
